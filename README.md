This project is a C implementation of a minimalist 3D rendering engine capable of displaying a rotating torus (donut) in the terminal, using only ASCII characters to simulate light and volume.

It is based on the famous code by Andy Sloane, deconstructed and rewritten to understand 3D geometry concepts, projection, and low-level memory management.

The theory explanation around the donut: https://www.a1k0n.net/2011/07/20/donut-math.html

> git clone git@github.com:ThibaultDausque/donut_math.git

> cc -Wall -Wextra -Werror donut.c -o donut
